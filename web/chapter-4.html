> قانون شماره ۱: هر مقدار در راست فقط یک مالک (Owner) دارد.

> اگر فریس تراکتورش را به دوستش «بیل» بدهد، دیگر خودش نمی‌تواند با آن بازی کند! به این کار انتقال مالکیت یا Move می‌گوییم.

// فریس مالک تراکتور است
let s1 = String::from("tractor");
// مالکیت به بیل (s2) منتقل می‌شود
let s2 = s1;
// println!("{}", s1); // ❌ Error! Value moved!

> Rule #1: Every value in Rust has exactly one owner.

> If Ferris gives his tractor to his friend Bill, Ferris can't play with it anymore! This is called Move.

// Ferris owns the tractor
let s1 = String::from("tractor");
// Ownership moves to Bill (s2)
let s2 = s1;
// println!("{}", s1); // ❌ Error! Value moved!

⚠️ تراکتور منتقل شد! فریس دیگر مالک آن نیست. ⚠️ Tractor moved! Ferris is no longer the owner.
