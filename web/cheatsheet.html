متغیرها (Variables)Variables

به صورت پیش‌فرض متغیرها غیرقابل تغییر (Immutable) هستند.

Variables are immutable by default.

let x = 5; // Immutable
let mut y = 10; // Mutable
y = 15;
const MAX_POINTS: u32 = 100_000;
توابع (Functions)Functions

آخرین خط بدون نقطه-ویرگول (;) مقدار خروجی را برمی‌گرداند.

The last line without a semicolon (;) is returned.

fn add(a: i32, b: i32) -> i32 {
    a + b // No semicolon!
}
مالکیت و امانت (Ownership)Ownership & Borrowing

انتقال مالکیت (Move) و امانت گرفتن (Borrowing).

Transferring ownership (Move) and Borrowing.

let s1 = String::from("hello");
let s2 = s1; // Move! s1 is invalid

let s3 = String::from("rust");
let r1 = &s3; // Immutable borrow

let mut s4 = String::from("crab");
let r2 = &mut s4; // Mutable borrow

Enums و MatchEnums & Match
enum Status {
    Ok,
    Error(String),
}

let state = Status::Ok;
match state {
    Status::Ok => println!("Good"),
    Status::Error(e) => println!("Err: {}", e),
}

مدیریت خطا (Errors)Error Handling
// Unrecoverable
panic!("Crash!");

// Recoverable (Result)
fn read() -> Result<String, io::Error> {
    let mut s = String::new();
    File::open("a.txt")?.read_to_string(&mut s)?;
    Ok(s)
}

    fn summarize(&self) -> String;
}

    println!("{}", item.summarize());
}
