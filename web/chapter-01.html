> Mission 1: To begin repairs, we must install the magic toolbox (rustup).

>> مرحله ۱: نصب rustup >> Step 1: Install rustup

جعبه ابزار (rustup) شامل کامپایلر (rustc) است. کامپایلر مثل یک مترجم، حرف‌های ما را به صفر و یک تبدیل می‌کند. روی دکمه زیر کلیک کن تا سیستم آن را دانلود کند:

The toolbox (rustup) contains the compiler (rustc). It translates our words into zeros and ones. Click below to download:

$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

>> مرحله ۲: برقراری ارتباط (Hello World) >> Step 2: Establish Connection (Hello World)

حالا باید به کامپیوتر ثابت کنیم که هستیم! باید پیام "Earth, I am here!" را روی صفحه چاپ کنیم. کلمه کلیدی درست را برای چاپ متن انتخاب کن:

>> مرحله ۳: دستیار هوشمند (Cargo) >> Step 3: The Smart Assistant (Cargo)

برای پروژه‌های واقعی، ما از دستیار هوشمند یعنی Cargo استفاده می‌کنیم. کارگو پروژه‌ها را می‌سازد و اجرا می‌کند.

For real projects, we use the smart assistant Cargo. Cargo builds and runs projects for us.

// ساخت پروژهCreate project
$ cargo new engine_start
$ cd engine_start
