> To build the Guess the Number game, we first need a random number.

> Computers don't have dice! We must add a toolbox called rand to our project.

// Adding the dice toolbox to our project
$ cargo add rand
// Generate a random number from 1 to 100
let secret_number = rand::thread_rng().gen_range(1..=100);
>> مرحله ۱: جعبه‌های حافظه (متغیرها) >> Step 1: Memory Boxes (Variables)

وقتی در Rust متغیری با let می‌سازیم، درِ آن قفل می‌شود و نمی‌توان محتوایش را تغییر داد (Immutable).

چون در این بازی بازیکن قرار است اعداد مختلفی را حدس بزند، باید قفل جعبه‌ی guess را باز بگذاریم.

برای ساخت یک متغیر قابل تغییر (Mutable)، کدام کلمه را باید اضافه کنیم؟

When we create a variable with let in Rust, it is locked and cannot be changed (Immutable).

Since the player will guess different numbers, we must leave the guess box unlocked.
